Spanning Tree Construction

Objective

Given a connected graph, the objective of this system is to build a spanning tree of the graph by selecting a subset of its nodes and edges.

Graph Color Codes

The table below, lists all the colors used in the graph, along with their interpretations.

Node/Edge Color Type
Unselected Node
Selected Node
Unselected Edge
Selected Edge

Setup

In this system, you are provided with a connected graph. You can click on any node/edge to add the node/edge (and connecting nodes) to the sub-graph.

At any point while building the sub-graph, you can click on any node/edge to remove it from the tree.
