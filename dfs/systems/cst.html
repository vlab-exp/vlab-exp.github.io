Graph Traversal: Spanning Tree

Objective

Given a connected graph, the objective of this system is to build a (any) spanning tree of the graph.

Setup

Initial: Selecting the first node

In this system, you are provided with a connected graph. You can click any node to select it. By selecting the first node you start building the spanning tree. Clicking on a selected node unselects it and removes the node from the tree.

Expanding the Spanning Tree

After selecting the first node, you will see spanning tree beside the graph. At this point the tree has just the one selected node. Now, you will observe that the edges connected to the selected node(s) are marked as "candidates" (See the color codes table).

You can select any of the candidate edges to add that edge to the spanning tree.

Once all the nodes are added to the tree, the spanning tree is complete and you will not see any candidate edges any more.

Backtracking

At any point while expanding the tree, you can click on a node that corresponds to a "leaf" in the spanning tree to remove the node from the tree.

Select any node to add it to the tree.
