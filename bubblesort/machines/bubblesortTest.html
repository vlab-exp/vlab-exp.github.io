Objective

In this experiment, you need to drive the the Bubblesort algorithm using the given controls, to sort the given list.

Experiment Setup

This experiment is similar to 'BubbleSort (Exercise)', with a different feedback system. The system does not provide any feedback, and tests your understanding. Once you complete the experiment then feedback is shown.

The experiment consits of a list of numbers and set of given controls. You can not click on the list element, but can only use the controls, to interact with the list. All the controls for performing Bubblesort are given.

  1. The 'Swap' button swaps the element at index 'i' with the element at index 'i+1'.
  2. The button 'Increment i' increments the value of 'i'.
  3. The control 'Reset i' resets the value of 'i' to 0.
  4. The 'Decrement b' button decrements the value of 'b' by 1. The two indicators 'i' and 'b', mark the values of 'current index' and 'boundary' variables involved in the algorithm.

Procedure

Follow the given steps to perform the experiment.

  1. Click a suitable control to simulte the next step of Bubblesort
  2. View Feedback and numbers, and repeat step 1, till you finish simulating bubblesort.
