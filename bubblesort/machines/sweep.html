Objective

Sort the given list by performing adjacent ordering operation using increment and reset to select the pairs.

Experiment Setup

The Bubble Machine invloves two control called 'Increment' and 'Reset'. The 'Increment' control allows you to iterate through the list, starting from index 0 going upto the last element. The 'Increment' control also performs adjacent ordering operation on the current index. The 'Reset' control takes the current index back to 0.

The fundamental operation in the Bubble Machine is the same as the Order Adjacent Machine, which is ordering adjacent numbers. The only difference is the way you select the pair to order.

Procedure

Follow the given steps to perform the experiment.

  1. Click on the 'Increment' button to go through the list.
  2. Hit 'Reset', to go back to the start.
  3. Check if the list is sorted or not.
